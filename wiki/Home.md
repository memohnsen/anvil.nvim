# Anvil documentation

Anvil is a 1:1 rewrite of Emacs' [Magit](https://magit.vc) and
[Forge](https://github.com/magit/forge) for Neovim — the full git porcelain
plus the GitHub/forge layer in a single plugin.

For installation and a quick start, see the [README](../README.md).

## Pages

- **[Configuration](Configuration.md)** — every `setup {}` option, with the full default config.
- **[Popups](Popups.md)** — the transient popup menus, argument state, and section navigation.
- **[Forge](Forge.md)** — the GitHub subsystem: sections, topic buffers, reviews, notifications.
- **[WIP Snapshots](WIP-Snapshots.md)** — Magit-style work-in-progress refs.
- **[Events](Events.md)** — the autocmd `User` events Anvil emits.

The full reference is also available in `:help anvil` inside Neovim.
